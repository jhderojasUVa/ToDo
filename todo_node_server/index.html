Simple ToDo Server

This is a very simple ToDo REST server. See what you can do:

  • /get: for getting all the TODOs
  • /get?id=1: for getting a ToDo by id
  • /post?whatToDo=XXXXX&completed=true|false: to add a ToDo
  • /delete?id=XXX: removes a ToDo by id
  • /update?id=XXX&whatToDo=YYYYYY&completed=true: updates a ToDo by id
  • /refactor: reorder all the ids on the ToDos

Every time you do the correct it will respond with the ToDos.
